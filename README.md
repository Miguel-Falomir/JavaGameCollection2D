# Java Game Collection 2D

A humble collection of 8 rip-offs of some of the most famous games in computers' history.
This project is merely for portfolio purposes, and focused on the process itself rather than the final result (hence the implementation of Swing instead of JavaFX).
All the features have been implemented in stock openjdk-17, without any kind of plugin or dependency.

## Overview

This program bundles a java version of these 8 games:
 + Solitaire (Klondike)
 + Game of Life
 + Snake
 + Pong
 + Breakout
 + Space Invaders
 + Tetris
 + Pacman

Also implements a settings menu that allows user to set language and theme.
Language can be set to:
 + English
 + Spanish
 + French (Not yet implemented at this moment)
Theme can be set to:
 + Metal
 + Nimbus
 + Motif
 + GTK
